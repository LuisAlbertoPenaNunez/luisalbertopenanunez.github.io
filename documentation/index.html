Introduction

TALENT is a One Page Personal HTML Template built with Bootstrap 3.3.6, HTML5 and CSS3. This template is well organized and valid code checked by W3C Validator. This Documentation will provide you a clear guideline to use and customize the template. Feel free to contact us if you need any further assistance. Thank you for purchasing our one page template.

Features

  • Responsive design
  • Modern & clean
  • W3C validated
  • Bootstrap 3.3.6
  • Google fonts
  • Themify Icon
  • Animated Text
  • Smooth scroll
  • Animation on reveal
  • Call to action
  • Ajax contact form
  • Well documented
  • Easy to customize
